HTML5 Raycaster Demo

This is a basic raycasting engine rendered on an HTML5 canvas.
See readme.txt for more information.

Code is available on GitHub
Old version with grid-based engine still available here.

Runs fastest in Chrome or IE9 and pretty slow in FF. Not yet tested in other browsers.
If you don't see anything, you should update your browser...

Use arrow keys to turn and walk, hold shift (or use < or >) to strafe.
